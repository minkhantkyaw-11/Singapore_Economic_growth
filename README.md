## Overview
This repository contains an academic data analysis project examining Singapore’s long-term economic growth using R.

## Objective
The project analyzes how factors such as capital accumulation, demographics, productivity, and policy relate to Singapore’s economic development using the Solow Growth Model.

## Data
- Penn World Table
- World Bank Open Data  
(Data files are not included.)

## Methods
Regression analysis and data visualization using R.

## Notes
This project was completed for learning purposes as part of university coursework.
